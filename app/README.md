# Dissertation AI App

Streamlit application for the MBA dissertation project:

**How Razorpay embedded finance affects Urban Company-style service providers in India**

## Structure

- `Home.py`: App landing page and navigation entry point.
- `pages/`: Streamlit multi-page dashboards.
- `components/`: Reusable UI components.
- `utils/`: Data, model, and report loading utilities.
- `models/`: Model wrapper and prediction helper code.
- `requirements.txt`: App-specific dependencies.

## Run

From the repository root:

```bash
streamlit run app/Home.py
```

This first step only creates the app skeleton. Data loading, model inference,
and charts will be added incrementally.
